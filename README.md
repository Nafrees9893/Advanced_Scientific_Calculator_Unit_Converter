# 🧮 Advanced Scientific Calculator & Unit Converter

A Windows-based scientific calculator and unit converter application developed in *C# (.NET 8.0, Windows Forms)*.  
This project integrates scientific computation, unit conversion, and temperature conversion into one user-friendly desktop app.

---

## 🚀 Features

- 🔢 *Scientific Calculator*
  - Trigonometric, logarithmic, and exponential functions
  - Parentheses handling and error validation

- 🔁 *Unit Converter*
  - Length, weight, and volume conversions
  - Dynamic form-based interface for quick switching between units

- 🌡 *Temperature Converter*
  - Convert between Celsius, Fahrenheit, and Kelvin

- 💻 *Multi-Form (MDI) Interface*
  - Smooth navigation between modules (Calculator, Unit Converter, Temperature Converter)

- 🎨 *Modern Blue-Themed UI*
  - Built using Windows Forms Designer in *Visual Studio 2022*
  - Simple, clear layout with splash screen for better user experience

---

## 🛠 Tech Stack

- *Language:* C#  
- *Framework:* .NET 8.0 (Windows Forms)  
- *IDE:* Visual Studio  
- *Interface:* MDI (Multiple Document Interface)
